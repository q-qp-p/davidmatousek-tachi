# Threat Infographic Design Template: Risk Reduction Funnel

> **Purpose**: Locked-in visual design for a 4-tier risk reduction funnel showing progressive
> pipeline risk reduction: threats identified -> inherent risk scored -> controls applied -> residual risk.
> The infographic agent MUST follow this layout. Only data values change between runs — structure stays fixed.

## Zone Specifications

### HEADER (~8% height)

- Title: "Risk Reduction Funnel" — 32px bold, #F8FAFC, left-aligned
- Project name: "{project_name}" — 16px regular, #94A3B8, below title
- Date: "{date}" — 16px regular, #94A3B8, right of project name
- CONFIDENTIAL badge: pill shape, #DC2626 background, #FFFFFF text, right-aligned

### FUNNEL (~62% height, ~75% width)

4 trapezoid tiers narrowing top-to-bottom, centered horizontally within the funnel zone:

**Tier 1 — Threats Identified (widest, 100% width)**
- Color: dominant severity color from threats.md Section 6
- Content: total finding count + qualitative severity distribution
- Label: "Threats Identified" left-aligned, white text
- Data: "{total_findings} findings — {critical}C / {high}H / {medium}M / {low}L"
- Data source: co-located threats.md Section 6 (Risk Summary)

**Tier 2 — Inherent Risk Scored (~75% width)**
- Color: dominant severity color from composite score distribution
- Content: composite score distribution and severity band counts
- Label: "Inherent Risk Scored" left-aligned, white text
- Data: "Composite range {min}-{max} — {critical}C / {high}H / {medium}M / {low}L"
- Data source: risk-scores.md Section 2 (Scored Threat Table)

**Tier 3 — Controls Applied (~50% width)**
- Color: dominant severity color based on control coverage
- Content: control coverage percentage, findings with controls count, mitigation statistics
- Label: "Controls Applied" left-aligned, white text
- Data: "{coverage}% coverage — {with_controls}/{total} findings mitigated"
- Data source: compensating-controls.md Section 1 (Executive Summary)
- **3-tier mode alternate label**: "Unmitigated Risk" (uses Tier 2 severity data)

**Tier 4 — Residual Risk (narrowest, ~30% width)**
- Color: dominant residual severity color
- Content: residual severity distribution and residual score range
- Label: "Residual Risk" left-aligned, white text
- Data: "Residual range {min}-{max} — {critical}C / {high}H / {medium}M / {low}L"
- Data source: compensating-controls.md Section 2 (Coverage Matrix)

**Tier Width Calculation**:
```
For each tier:
  actual_width = (tier_volume / tier_1_volume) * 100
  min_width = previous_tier_width - 10    # Minimum 10% narrowing per tier
  tier_width = max(actual_width, min_width)
  tier_width = max(tier_width, 10)         # Absolute floor: 10%
```

**Gradient Connectors**: Between each adjacent tier pair, render a gradient transition blending the upper tier's color into the lower tier's color. Connectors are ~3% of funnel height, creating a smooth flowing visual rather than hard edges.

**Ghost Tier Rendering** (when data source unavailable):
- Border: 2px dashed #475569 (Slate-600)
- Fill: #475569 at 20% opacity
- CTA text: centered inside tier, white (#FFFFFF), 14px regular
- Ghost tiers maintain the same width as if data were present (funnel shape preserved)
- No data content — only the CTA label

Ghost tier CTA labels by mode:
- **3-tier mode** (risk-scores source): Tier 4 ghost CTA = "Run /compensating-controls to complete the funnel"
- **1-tier mode** (threats source): Tier 2 ghost CTA = "Run /risk-score", Tier 3 ghost CTA = "Run /compensating-controls", Tier 4 ghost CTA = "Complete the pipeline"

### METRICS SIDEBAR (~20% width, right-aligned)

Right-aligned panel alongside the funnel zone. Dark card background (#334155), 12px rounded corners.

Content varies by data source mode:

**4-tier mode** (compensating-controls source):
- Total Findings: "{total_findings}"
- Risk Reduction: "{risk_reduction_pct}%"
- Control Coverage: "{control_coverage_pct}%"
- Severity breakdown per tier with colored indicators
- Percentage annotations between tiers showing stage-to-stage reduction

**3-tier mode** (risk-scores source):
- Total Findings: "{total_findings}"
- Severity Distribution: per-severity counts with colored indicators
- Risk Reduction: "N/A — run /compensating-controls"

**1-tier mode** (threats source):
- Total Findings: "{total_findings}"
- Qualitative severity counts only (Critical, High, Medium, Low)

### FOOTER (~5% height)

- Text: "Generated by Tachi Threat Modeling Framework — Risk Reduction Funnel"
- 12px regular, #94A3B8, centered

## Accessibility

- All severity levels distinguishable by label AND color (not color alone)
- Minimum contrast ratio 4.5:1 for text on dark backgrounds
- Tier labels provide text-based identification independent of color
- Ghost tiers use both dashed border AND CTA text for distinction (not opacity alone)
- Metrics sidebar uses text labels alongside colored indicators
- Severity counts in tier data provide numeric identification
