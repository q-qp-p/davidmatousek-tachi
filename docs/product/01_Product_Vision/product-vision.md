# Product Vision — tachi

## Mission Statement
Automated threat modeling toolkit extending STRIDE with AI-specific threat agents for agentic applications

## Vision Statement
The default threat modeling toolkit for any team building agentic AI applications

## Core Value Proposition
First toolkit to natively model AI-specific threat agents (prompt injection, tool misuse, agent autonomy risks) alongside traditional STRIDE

## Target Users
Developers building AI agents who need to understand security implications without deep security expertise

## Success Metrics
- GitHub stars, downloads, and active projects using tachi as their threat modeling tool
