# Product Vision — tachi

## Mission Statement
Threat modeling and AI-reasoning vulnerability detection harness for Claude Code, extending STRIDE with AI-specific threat agents and MAESTRO layer classification for agentic applications.

## Vision Statement
The default threat modeling and AI-reasoning vulnerability detection harness for any team building agentic AI applications — a new scanning column alongside SAST, SCA, and Secrets.

## Core Value Proposition
First harness to reason over architecture descriptions for both traditional threat modeling (STRIDE) and AI-specific threats (prompt injection, tool misuse, agent autonomy, human-trust exploitation). SAST catches syntax-level bugs; tachi catches logic-level ones.

## Target Users
Developers building AI agents who need to understand security implications without deep security expertise

## Success Metrics
- GitHub stars, downloads, and active projects using tachi as their threat modeling and vulnerability detection harness
