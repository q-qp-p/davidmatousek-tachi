## Zone Specifications

### TOP SECTION (~10% height)

- Title: "MAESTRO Layer Risk Distribution: {project_name}" — 32px bold
- Date: "{date}" — 16px regular, right of title
- CONFIDENTIAL badge: red background, white text, pill shape, right-aligned
- Subtitle: "{total_findings} Findings Across {layer_count} Layers"

### STACK ZONE (~70% height, ~70% width)

**Seven Horizontal Bands:**
- Stacked vertically: L7 (top) through L1 (bottom), following CSA MAESTRO layer ordering
- Each band spans the full width of the stack zone
- Band height: equal distribution across the 7 layers within the zone

**Per-Band Content:**
- Left side: layer ID and name (e.g., "L7 — Agent Ecosystem") — 16px semibold, white text
- Center: finding count badge — rounded pill showing integer count, severity-colored background
- Center-right: highest severity indicator — colored dot or icon matching the most severe finding in that layer
- Right side: up to 2 top finding summaries — format: "{ID}: {short threat}" in 12px regular

**Most-Exposed Layer (visual prominence):**
- The layer with the highest finding count receives visual emphasis
- Wider left border accent (6px vs 2px) in the severity color of its highest finding
- Brighter background (#475569 Slate-600 instead of standard #334155 Slate-700)
- Subtle glow or highlight effect to draw the eye

**Empty Layers (zero findings):**
- Still rendered as a band in the stack (all 7 layers always visible)
- Muted appearance: darker background (#1E293B matching dashboard background)
- Grayed text (#64748B Slate-500) for layer name
- No finding count badge, no severity indicator
- Dash (—) where finding summaries would appear

### SIDEBAR (~30% width)

**Aggregate Statistics:**
- "Total Findings: {N}" — 20px bold, white text
- "Layers with Findings: {N}" — 14px regular, secondary text
- "Empty Layers: {N}" — 14px regular, secondary text

**Most Exposed Layer Badge:**
- Prominent card within the sidebar
- Title: "MOST EXPOSED" — 14px bold, uppercase, white text
- Layer identifier: "{layer_id} — {layer_name}" — 16px semibold
- Finding count: "{finding_count} findings" — 20px bold, severity-colored
- Left border accent matching the severity color of the layer's highest finding
- Subtle card shadow for depth

**Severity Legend:**
- Below the most exposed badge
- Four entries with colored dots:
  - Critical (red dot)
  - High (orange dot)
  - Medium (amber dot)
  - Low (blue dot)
- Each entry: colored dot + severity name — 14px regular

### FOOTER (~5% height)

- "Generated by Tachi Threat Modeling Framework — CSA MAESTRO Layer Analysis"
- 12px, Gray-500, centered

## Accessibility

- All seven layer labels readable with text names alongside layer IDs (not ID alone)
- Color is not the sole severity indicator — text labels ("Critical", "High", "Medium", "Low") accompany all color-coded elements
- Finding count badges use numeric text alongside colored backgrounds
- Empty layers identified by text dash (—) in addition to muted visual styling
- Minimum contrast ratio 4.5:1 for text on backgrounds
- Most-exposed layer identifiable by border width and background brightness, not color alone
